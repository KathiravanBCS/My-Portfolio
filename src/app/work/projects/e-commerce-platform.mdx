## Overview

A complete e-commerce platform built from the ground up with a React frontend and Node.js backend. Features include product browsing, cart management, secure checkout, and payment processing.

## Key Features

- **Product Catalog**: Dynamic product listings with search, filtering, and category navigation
- **Shopping Cart**: Persistent cart with real-time quantity updates and price calculations
- **Secure Checkout**: Payment integration with Stripe/PayPal for safe transactions
- **Responsive Design**: Mobile-first UI that works across all screen sizes

## Technologies Used

- **React**: Component-based frontend with state management
- **Node.js**: RESTful API backend for product and order management
- **SQL**: Relational database for products, users, and orders
- **Stripe/PayPal**: Secure payment processing integration

## Outcome

A fully functional e-commerce platform demonstrating end-to-end full-stack development skills from database design to payment integration.
