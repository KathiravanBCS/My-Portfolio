## Overview

A dynamic movie discovery application that lets users explore, search, and filter movies using real-time data from the TMDB API. Features a clean, responsive interface with detailed movie information.

## Key Features

- **Real-Time Movie Data**: Integration with the TMDB API for up-to-date movie information
- **Search & Filtering**: Instant search with genre and popularity filtering
- **Movie Details**: Rich detail pages with ratings, cast, and related movies
- **Responsive UI**: Optimized layout for both desktop and mobile browsing

## Technologies Used

- **React.js**: Frontend framework for the interactive UI
- **Node.js**: Backend proxy for API key management
- **TMDB API**: The Movie Database API for movie data
- **MongoDB**: Database for user preferences and watchlists
- **AWS**: Cloud hosting and deployment

## Outcome

A polished movie discovery experience demonstrating API integration skills, responsive design, and full-stack deployment on AWS.
