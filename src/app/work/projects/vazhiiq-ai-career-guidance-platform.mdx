## Overview

VazhiIQ is an AI-powered career guidance platform that helps users navigate their professional journey with intelligent recommendations. The name "Vazhi" (வழி) means "path" in Tamil — reflecting the platform's mission to guide users on their career path.

## Key Features

- **Custom AI Prompt System**: Built a structured Groq API integration with carefully engineered prompts for accurate career recommendations
- **Gamification Dashboard**: Progress tracking with rewards system to keep users engaged in their career development journey
- **Career Summary Generation**: Automated career summaries powered by Large Language Models
- **Brand Identity Design**: Designed a full brand identity including logo and comprehensive design system

## Technologies Used

- **React**: Component-based UI architecture
- **TypeScript**: Type-safe development across the entire codebase
- **Groq API**: Fast AI inference for career guidance features
- **Mantine UI**: Modern, accessible component library for the interface
- **Recharts**: Data visualization for the gamification dashboard

## Challenges and Learnings

Building an effective AI prompt system required extensive iteration to ensure career recommendations were accurate and personalized. The gamification layer needed careful balance — engaging enough to drive usage without feeling gimmicky.

## Outcome

VazhiIQ demonstrates the potential of AI-driven career guidance, providing personalized recommendations that help users make informed decisions about their professional development.
