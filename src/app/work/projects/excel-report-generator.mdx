## Overview

A robust Excel report generation system designed for enterprise-grade data exports. The system handles dynamic row insertion, formula adjustment, and produces audit-ready reports with professional styling.

## Key Features

- **AutoFilter & Table Metadata Repair**: Handles large file edge cases where Excel metadata breaks
- **Regex-Driven Formula Adjustment**: Automatically shifts formula references when rows are inserted
- **Locked/Unlocked Cell Styling**: Cell-level protection with validation support for data entry
- **Snapshot Stamping**: Audit-ready report generation with timestamp and metadata embedding

## Technologies Used

- **TypeScript**: Type-safe template processing logic
- **ExcelJS**: Programmatic Excel file generation and manipulation
- **Node.js**: Server-side generation for large dataset processing

## Challenges and Learnings

Working with the ExcelJS library at scale revealed edge cases around AutoFilter metadata corruption and formula reference shifting. Building a regex-based formula adjuster that handles all Excel formula patterns was particularly challenging.

## Outcome

The system generates production reports with perfect fidelity, handling datasets of thousands of rows while maintaining proper formatting, formulas, and cell protection.
