## Overview

Travel Buddy is a comprehensive UI/UX design project for a map-first mobile travel application. The design emphasizes intuitive navigation, rich visual hierarchy, and a cohesive design language.

## Key Features

- **Design System**: Full design system with teal/amber color tokens for a warm, travel-inspired palette
- **Map-First Screen Flows**: Navigation patterns centered around map interactions
- **Mobile-Responsive Components**: A component library optimized for touch interactions and various screen sizes

## Design Approach

The design prioritizes the map as the primary interface element, letting users discover places visually. The teal/amber color palette evokes a sense of adventure while maintaining readability and accessibility standards.

## Tools Used

- **Figma**: For creating and managing all design assets, prototypes, and the component library

## Outcome

The design delivers a polished, production-ready UI kit that can be handed off to developers with clear specifications, consistent spacing, and well-documented interaction patterns.
