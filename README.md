
# Portfolio Project

This is a simple, clean, beginner-friendly portfolio template built with **Next.js** and **Tailwind CSS**. It supports MDX-based content for projects and blog posts, an about/CV page, and a gallery.

## Getting started


**1. Clone the repository**
```
git clone <your-repo-url>
```

**2. Install dependencies**
```
npm install
```

**3. Run dev server**
```
npm run dev
```


**4. Edit content**
```
src/resources/content.tsx
```

**5. Create blog posts / projects**
```
Add a new .mdx file to src/app/blog/posts or src/app/work/projects
```

This project is built with [Next.js](https://nextjs.org) and [Tailwind CSS](https://tailwindcss.com). It requires Node.js v18.17+.


## Documentation

See the official docs for [Next.js](https://nextjs.org/docs) and [Tailwind CSS](https://tailwindcss.com/docs) for more information.

## Features


### Features

- Built with [Next.js](https://nextjs.org) and [Tailwind CSS](https://tailwindcss.com)
- MDX-based content system for projects and blog posts
- About/CV page and gallery
- Responsive layout optimized for all screen sizes
- SEO-friendly with customizable metadata
- Easy to customize and extend
